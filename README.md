# Claude Quick Access

Minimal Chrome Extension based on Manifest V3.

## What it does

- adds a toolbar action button
- opens `https://claude.ai/` on click
- reuses an existing Claude tab when one is already open

## Files

- `manifest.json` - extension configuration
- `background.js` - action click logic
- `icons/` - placeholder toolbar/store icons

## Load in Chrome

1. Open `chrome://extensions/`
2. Enable Developer mode
3. Click `Load unpacked`
4. Select this folder
